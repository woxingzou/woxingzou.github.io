智能AI微信机器人

709 字约 2 分钟

wechatyAI

2025-01-09

昨天在github上发现了一个很好玩的项目WeChat Bot，一个 基于 chatgpt + wechaty 的微信机器人，可以用来帮助你自动回复微信消息，或者管理微信群/好友。但是使用的wechatyAPI是微信网页版，该机器人不被微信官方支持，所以有封号风险，请谨慎使用（开个小号玩玩就行，但是好像只能是2017年之前创建的微信号才行）。

下载项目

git clone --depth 1 https://github.com/wangrongding/wechat-bot.git

修改配置文件

项目支持多种AI大模型，你可以选择一款进行尝试，推荐deepseek（当前很便宜，10块钱500万token，个人使用够呛能用的完），点我查看

修改项目根目录的.env配置，如果没有可以直接拷贝一份.env.example

...
# deepseek-free, model必须为deepseek-chat或deepseek-coder,去 https://platform.deepseek.com/usage或者https://github.com/LLM-Red-Team/deepseek-free-api
# 在DEEPSEEK_SYSTEM_MESSAGE中设置系统提示词
DEEPSEEK_FREE_URL='https://api.deepseek.com/chat/completions'
DEEPSEEK_FREE_TOKEN='xxxxxxxxxxx' # 修改为你可apikey
DEEPSEEK_FREE_MODEL='deepseek-chat'
DEEPSEEK_SYSTEM_MESSAGE='You are a personal assistant.'
...
# 白名单配置
#定义机器人的名称，这里是为了防止群聊消息太多，所以只有艾特机器人才会回复，
#这里不要把@去掉，在@后面加上你启动机器人账号的微信名称
BOT_NAME='@智能AI助手'
#联系人白名单
ALIAS_WHITELIST=''
#群聊白名单，添加需要自动回复的群聊名称
ROOM_WHITELIST=''
#自动回复前缀匹配，文本消息匹配到指定前缀时，才会触发自动回复，不配或配空串情况下该配置不生效（适用于用大号，不期望每次被@或者私聊时都触发自动回复的人群）
#匹配规则：群聊消息去掉${BOT_NAME}并trim后进行前缀匹配，私聊消息trim后直接进行前缀匹配
AUTO_REPLY_PREFIX=''

# 默认服务 302AI，ChatGPT、Kimi、Xunfei、deepseek-free, ollama, dify, tongyi 八选一，不填则键盘交互
SERVICE_TYPE='deepseek-free'

安装依赖&运行服务

安装依赖

# 切换淘宝镜像
npm config set registry https://registry.npmmirror.com
# 安装依赖
npm i
# 推荐用 yarn 吧，npm 安装有时会遇到 wechaty 内部依赖安装失败的问题
yarn

运行服务

# 启动服务
npm run dev # 或者 npm run start
# 启动服务
yarn dev # 或者 yarn start

服务启动后会提示你扫码登录，然后就可以使用AI大模型自动回复在群聊里主动@你的消息了

结尾

该机器人由于使用微信网页版API接口不被微信官方支持，因此存在一定的封号风险，请谨慎使用
