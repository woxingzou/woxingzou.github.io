新增打赏功能

418 字约 1 分钟

None

2025-01-10

简介

使用主题自带的布局插槽给博客添加打赏功能 参考文档

AI这么好用肯定要利用一下

直接询问deepseek

使用vue3组合式语法帮我写一个打赏按钮，效果是鼠标放置在按钮上时上方显示两个二维码图片，script标签在最上方，style使用scss语法，按钮居中展示

deepseek的回答

<script setup>
import { ref } from 'vue';

// 控制二维码显示状态
const showQRCode = ref(false);
</script>

<template>
  <div class="donation-container">
    <!-- 打赏按钮 -->
    <button
      class="donation-button"
      @mouseenter="showQRCode = true"
      @mouseleave="showQRCode = false"
    >
      打赏
    </button>

    <!-- 二维码区域 -->
    <div v-if="showQRCode" class="qrcode-container">
      <img src="your qr code image path" alt="QR Code 1" class="qrcode" />
      <img src="your qr code image path" alt="QR Code 2" class="qrcode" />
    </div>
  </div>
</template>

<style lang="scss" scoped>
.donation-container {
  position: relative;
  display: flex;
  justify-content: center;
  align-items: center;
}

.donation-button {
  padding: 10px 20px;
  font-size: 16px;
  color: #fff;
  background-color: #007bff;
  border: none;
  border-radius: 5px;
  cursor: pointer;
  transition: background-color 0.3s ease;

  &:hover {
    background-color: #0056b3;
  }
}

.qrcode-container {
  position: absolute;
  bottom: 100%; // 将二维码放置在按钮上方
  z-index: 1000; // 确保二维码在最上层
  display: flex;
  gap: 10px;
  padding: 10px;
  background-color: #fff;
  border: 1px solid #ddd;
  border-radius: 5px;
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
}

.qrcode {
  width: 280px;
  height: 290px;
  border-radius: 5px;
}
</style>

提示

  • 记得修改21和22行的二维码图片路径，当然你也可以添加多个

在docs/.vuepress/theme/components目录下新增Donation.vue文件，把上面的代码复制进去

修改docs/.vuepress/client.ts文件

import { h } from 'vue'
import { Layout } from 'vuepress-theme-plume/client'
import { defineClientConfig } from 'vuepress/client'
import Donation from './theme/components/Donation.vue'

export default defineClientConfig({
  ...
  layouts: {
    Layout: () => h(Layout, null, {
      'doc-footer-before': () => h(Donation),
    }),
  },
  ...
})
